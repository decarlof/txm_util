Usage;

rec.py: reconstruct original raw data 

help:  python rec.py -h
usage: python rec.py /local/dataraid/mark/Run2/radios/ 1


align.py: alignes the original raw data

help:  python align.py -h
usage: python align.py /local/dataraid/mark/Run2/radios/ 1


rec_aligned.py: reconstruct the aligned data

help:  python rec_aligned.py -h
usage: python rec_aligned.py /local/decarlo/conda/util/mark/aligned/blur_iter_20_bin_2/radios/ 0

plot.py: plot x-y corrections
